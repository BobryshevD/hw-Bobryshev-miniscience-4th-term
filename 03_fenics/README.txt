Первый файл - с решением уравнения Пуассона в виде u = 1 + x^2 + y^2: функция на границе u_0 = 1 + x^2 + y^2, f = -4, 
граница - весь квадрат. Здесь условия Неймана не влияют на решение, т.к. при фиксированном y задача сводится к задаче u'' = const,
т. е. u = ax^2 + bx + c. Задача полностью определяется значением на нижней границе, значением на верхней границе и const. (Граница = край)
Ясно, что если оставить только нижнюю границу квадрата, условие Неймана станет существенным.
Второй файл - с решением уравнения Пуассона в виде u = sin(5x) + sin(5y).
Третий файл - граница x = 0, т.е. только левая сторона квадрата. Значение функции на границе u_0 = 1; производная меняется по закону
u' = sin(5y); Функция f = 10*exp(-(dx*dx + dy*dy)/ 0.02).(dx = x-0.5, dy = y-0.5)
Четвёртый файл - граница x=0; Значение функции u_0 = 1; Функция u' = 10*exp(-(x*x + y*y)/ 0.02) на границе; Функция f = 10*exp(-(x*x + y*y)/ 0.02).
Достаточно красочная картинка.
Интересное поле с какой-то нетривиальной границей(например, для x = 1/2) придумать не удалось.
